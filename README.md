## Features
- Save any piece of important information to you.
- Fast search to find items instantly by title, notes, or content.
- One-tap copy to clipboard.
- Organize with drawers (smart filters supported).
- Privacy-first: on-device encryption and no analytics.

## Getting Started
- Requirements:
  - iOS 26 or later
  - Xcode 26 or later
- Build & Run:
  1. Open the project in Xcode.
  2. Select the Cabinet app target.
  3. Press Run (âŒ˜R) to build and launch on a device or simulator.

## Usage
- Create a new item: Tap the + button and enter a title, content, and optional notes.
- Search: Use the search bar to filter your items.
- Copy: Tap an item to copy its content.
- Organize: Use the swipe actions or context menu to manage items.

## Privacy & Security
- Data is stored securely on-device.
- Sensitive fields can be masked by default and revealed on demand.
- We do not collect personal analytics by default.
